Chat_Application
💬 Chat Application – Python Socket Programming
📌 Project Overview
This is a simple client-server Chat Application built using Python socket programming. It enables real-time communication between a client and a server using TCP connection on localhost.
🚀 Features
Real-time communication between client and server
Username-based chat system
Continuous message exchange
Exit command to end chat safely
Simple terminal-based interface
🛠️ Technologies Used
Python
Socket Programming (TCP/IP)
Command Line Interface
⚙️ How It Works
Server starts and waits for client connection
Client connects using IP address and port number
Client sends username to server
Messages are exchanged in real time
Chat ends when user types "exit"
▶️ How to Run
Run server first: python server.py Run client: python client.py
💡 Example Output
Client: Enter your name: Amulya
You: Hello
Server: Hi
Server: Amulya joined the chat
Amulya: Hello
You: Hi
⚠️ Notes
Start server before client
Works on localhost (127.0.0.1)
Uses port 5000
